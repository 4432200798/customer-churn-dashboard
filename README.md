# Customer Churn Dashboard 📊
Interactive Power BI dashboard designed to analyze customer churn behavior and identify key factors affecting customer retention.
## Dashboard Insights
- Churn Rate Analysis
- Internet Service Comparison
- Contract Type Analysis
- Payment Method Trends
- Customer Tenure Analysis

## Tools Used
- Power BI
- Data Visualization
- Data Analysis
